# Bug Tracking Log

*See `.github/copilot-instructions.md` for the Bug Protocol.*

## Priority Definitions

| Priority | Definition | Response SLA | Examples |
|----------|------------|--------------|----------|
| **P0** | Critical outage, production down | Immediate (drop everything) | Service unavailable, data corruption, security breach |
| **P1** | Major functionality broken | Within 24 hours | Core feature non-functional, blocking user workflows |
| **P2** | Significant issue with workaround | Within 1 sprint | Feature degraded but usable, intermittent failures |
| **P3** | Minor/cosmetic issue | Backlog | UI polish, non-critical edge cases, minor UX issues |

## Status Definitions

| Status | Description |
|--------|-------------|
| **New** | Reported but not yet investigated |
| **Confirmed** | Reproduced and validated |
| **In Progress** | Actively being fixed |
| **Fixed** | Code change merged |
| **Closed** | Verified in production |
