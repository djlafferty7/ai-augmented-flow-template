## Backlog Structure Guide

### Status Values
- **Pending** — Not started
- **In Progress** — Actively being worked
- **Blocked** — Waiting on dependency
- **✅ Complete** — Done

### Estimation
- Use days (Xd) for rough sizing
- 1d = ~6 hours of focused work

### Linking
- Reference PRDs: `(PRD: [PRD-001](prd/PRD-001-slug.md))`
- Reference Tech Debt: `(Ref: TECHNICAL_DEBT.md DEBT-XXX)`
- Reference Bugs: `(Ref: BUGS.md BUG-XXX)`
